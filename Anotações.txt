ctrl + F5	inicia o código semdepuração
F5	inicia o código com depuração
ctrl + k + d	organiza o código
ctrl + d 	duplica a linha onde o cursor está
ctrl + F	realiza busca no documento atual
outros	https://visualstudio.microsoft.com/keyboard-shortcuts.pdf


para criar novo projeto na mesma solution
	- botão direto na solution
	- adicionar > novo projeto

para criar uma classe
	- ctrl + n
	- selecionar : Classe C# 
	
para condigurar a execução de mais de um projeto na solution
	- botão direito na solution
	- propriedades
	- Na janela da esquerda: Propriedades Comuns > Projeto de inicialização
	- Na Janela da direita:
		.Seleção atual (recomendado) : executa o projeto selecionado
		. Único projeto: define apenas um projero que será executado
		. Vários projetos: executa um ou mais projetos em cascata, de acordo com a ordem.
